# 밀키트 전문점

# 주요 서비스
1) 고객이 원하는 수량의 메뉴를 담아 결제를 지원합니다.
2) 관리자는 물품을 등록,수정,삭제 할 수 있습니다.
3) 관리자페이지는 로그인을 통해서만 접근 할 수 있다.
4) 물품에 대해서 매진 또는 판매를 설정 할 수 있다.
